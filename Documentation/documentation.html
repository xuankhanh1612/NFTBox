About NFTBOX

NFTBOX is the complete software solution for your Non-Fungible Token (NFT) Marketplace Business. Capable of minting NFTs over the blockchain whilst allowing consumers to Buy, Sell and Transfer their NFTs on the marketplace freely with indisputable proof of ownership and authenticity achieved using ERC-721 standard smart contracts. Additionally, NFTBOX also incorporates all the necessary features needed for a present-day NFT marketplace such as a bidding system, and NFT collections as well as granting creators the option to sell and trade their digital works in various forms such as images, GIFs, video, and audio.

Key Features of NFTBOX

  1. The Admin Panel dashboard is a hub where admins can effortlessly get information about:
    • Total active users in the marketplace.
    • Total NFTs minted in the marketplace.
    • Total NFTs listed for sale.
    • Total NFTs bought and sold in the marketplace.
    • Total inactive users in the marketplace.
    • Summary report of the total NFTs sold sorted by month and year in a graphical form.
    • Summary report of the total NFTs sold, total NFTs listed for sell and total NFTs minted all filterable by year in graphical view.
  2. Admin can add sub-admins to manage the marketplace software and also get information such as the last login and logout times along with the active and inactive statuses of all administrators.
  3. Admins can view all marketplace users information and can check on their activities.
  4. Through the admin panel, admins can view and manage the list of all NFTs minted on the marketplace as well as see their details, information such as NFT name, token ID, owner ID, owner name, NFT owner wallet address as well as the status of the NFT, etc.
  5. Admin wallet can be setup from the admin panel with ease.
  6. Admins can deploy their marketplace smart contract with ease directly from the admin panel.
  7. From the admin panel, admins can set the network information for the marketplace such as the network name, Chain ID, symbol, and RPC.
  8. In this system, you can edit the website's About, Contact, Terms, Privacy policy, F.A.Q and Social links section from the Conetent Manager System section present in the Admin panel.

Requirements for NFTBOX System

In this section we will present the recommended configuration under which NFTBOX would work perfectly. These are not absolute values as the performance depends on many factors such as host performance, server configuration and the number of visitors.

Server requirements

  • Apache or IIS or Nginx
  • PHP version 7.4 > 8.0 (ie. recommended: Any PHP version greater than 7.4 and less than 8.0)
  • MySQL version 5.0 or greater (recommended: MySQL 5.5 or greater)
  • Node.js version 16.15.0 (LTS)
  • We recommend DigitalOcean for hosting.

NFTBOX Installation

  1. Please extract the NFTBOX_Main-v-1.0.zip folder. You will then get the following:

    • NFTBOX_Main-v-1.0.zip
    • Documentation (Folder)

    We highly recommend that you read the documentation first.

  2. Upload the NFTBOX_Main-v-1.0.zip file to your server using cPanel or FTP.
  3. Extract the NFTBOX_Main-v-1.0.zip file.
  4. Navigate into the extracted folder and then move all the folders and files present (eg. application, assets, index.php, install, and system folders) one directory up ie. into the public_html directory. Alternatively, you could create a comment folder and then move the files into that comment folder.
    CAUTION: Make sure that the .htaccess file is moved correctly or else you will be unable to access the admin panel.

    5. NOTE: The instructions above will be applicable for all other system!

  5. To setup your application, browse: http://yourdomain.com/install . Then enter your Envato User ID and Purchase Key. (NOTE: Your purchase key can be found at your envato download option: https://themeforest.net/downloads) Once the form has been filled, click on the Next button.

  6. Installer second step checks to see whether your directory and server meet the application installation requirements.
    NOTE: SMTP server must be enabled to send emails. However, it is not mandatory during the installation process. You will have to setup your SMTP server later otherwise your emails will fail to deliver.


    7. IMPORTANT NOTE: If you see errors in "Directory permissions & requirements" due to files not being writable as shown in the screenshot below then please change their read-write permissions in the file properties as guided by the screenshots below.

    • install.sql (Location: /var/www/html/install/sql)

    • Database.php (Location: /var/www/html/app/Config)

    • Database.php (Location: /var/www/html/install/php)

    • App.php (Location: /var/www/html/app/Config)

    • captcha (folder) (Location: /var/www/html/public/assets/images)

    • session (folder) (Location: /var/www/html/writable)

    • cache (folder) (Location: /var/www/html/writable)

    • uploads (folder) (Location: /var/www/html/public)

    • flag (folder) (Location: /var/www/html/install)


  7. Enter your Database Name, Username, Password and Host Name then click on the Next button.

  8. Then wait for 60 seconds.

  9. Now use a temporary Username and Password to login to the system.
    NOTE: Please change this login information (Username and Password) after you are logged in.

    10. Installation successfully done!
  10. Now, Click on Click to launch your application button to access your application.
  11. Thank you for purchasing NFTBOX. If you have any questions, please contact us at info@bdtask.com.

    6. NFT Setup - NFT Marketplace Blockchain Setup
      NOTE: Make sure your NodeJS server is running as you will need the port number during NFT marketplace setup.

      The default network set is the Polygon Mumbai Testnet. This can be changed to Polygon Mainnet by clicking on the Update button.



      Then updating the fields using the Polygon Mainnet setting. (NOTE: These Polygon Mainnet settings can be found by clicking on the "Polygon Technology" link provided at the top of the form as shown below.

      Network Name: Should be set to an appropriate name that can be used to easily identify the name of the blockchain network the NFT Marketplace software is connected to later on. We recommend that you set Network Name to “Smart Chain” as suggested in Binance Academy

      Chain ID: It is used for replay protection of transactions preventing attacks between the chains with same NetworkID’s. Signed Transactions will have a different hash due to the addition of the chain ID along with the previous standard of only using Network IDs. You must set the Chain ID to 56 as stated in Binance Academy

      Current Symbol: Binance Coin is the cryptocurrency issued by Binance exchange and trades with the BNB symbol. You must set Currency Symbol to “BNB” as stated in Binance Academy

      RPC URL: RPC(Remote Procedure Call) is a set of protocols and interfaces that the client interacts with blockchain system. RPC URL is a URL to which requests for blockchain data can be sent. (Similar to RPC Endpoint which is the node’s address). The Blockchain related information can be queried after. (Such as block number, blocks, node connection, etc.) and send the transaction request through RPC interface. For updated BNB mainnet RPC URL go to Binance Academy

      Block Explorer: A block explorer is an online tool that enables you to search for real-time and historical information about a blockchain, including data related to blocks, transactions, addresses, and more. We recommend that you set Block Explorer URL to “https://bscscan.com” as stated in Binance Academy

      Server IP: The IP of your domain server.

      Port: You can choose any open ports that are not used by any other services.

    7. NFT Setup - NFT Marketplace Contract Deploy

      To deploy your NFT Marketplace smart contract go to: NFT Setup > NFT Setup.

      IMPORTANT NOTE: The contract deploy function will no longer be visible once a contract has been deployed. And the hash of the contract transaction will appear here.



      Contract Name: Preferably the name of your NFT Marketplace.

      Token Symbol: Token used to mint NFTs in your NFT marketplace must be given a symbol. (eg. BTC, ETH, BNB etc.)

      Max Supply: This is the max NFT minting limit you want to initially set for your NFT Marketplace. (Once the max limit is reached then no more NFT’s can be minted!). You can increase the max supply limit whenever you wish after deploying the smart contract. (Note: You will need to pay the gas cost for the transaction)


      8. After successfully deploying your NFT Marketplace smart contract you can see the deployed smart contract information such as Contract Address and Transaction here. To see more information you can click on the paper plane icon buttons and it will lead you to their Etherscan address.




      IMPORTANT: NFTBOX Smart Contract Verification


      (1) Go to Admin Panel > Contract Deploy and then click on Contract Address


      (2) Click on Contract.


      (3) Click on Verify and Publish.


      (4) For the Compiler Type please click on Solidity (Single file) from the dropdown.


      (5) For Compiler Verion please click on v0.8.4+commit.c7e474f2 from the dropdown.


      (6) For Compiler Verion please click on v0.8.4+commit.c7e474f2 from the dropdown.


      (7) Agree to the Terms of service then click on Continue


      (8) Copy all of the text from the text file (.txt) provided with NFTBOX.



      (9) Paste the copied text in the Enter the Solidity Contract Code below* text box.


      (10) Pass the I'm not a robot test and then click on the Verify and Publish button.


      You should now have a successfully verified marketplace smart contract!


      To confirm that your smart contract has been verified you can go to Contract Deploy > Contract Address (as shown in the screeshot below) and you should be able to see the smart contract in the Contract section along with the green verified check mark indicating that the smart contract has been verified.



      Now for every transaction that takes place over the blockchain from your marketplace, you should be able to see what event took place by looking under the Method section.

        Customers can manage their settings by clicking on their Profile Icon in the top right of the homepage and then selecting Profile.


        Profile - Edit
        Here customers can edit their profile information such as their First Name, Last Name, Bio, Personal Site or Portfolio and their Wallet Address.



        Profile - Withdraw
        (1) Here the customer can withdraw their marketplace funds.
        (2) Customers must click on Reload Balance to check their current updated balance.
        (3) Here the customer must enter the amount they want to withdraw
        (4) Finally to commit their withdraw action they must press the Send button.



        Profile - Transactions
        (1) The customer can check all of the previous transacitons on the marketplace that took place over the blockchain by going to the Transactions seciton.
        (2) To see more information about each individual transactions they can click on Trx details.

        Customers can see their owned NFT's, favorite NFTs and NFT Collections by clicking on their Profile Icon in the top right of the homepage and then selecting My NFT's.




      4. My Collections

        (1) Customers can see their Collections by clicking on their Profile Icon in the top right of the homepage and then selecting My Collections.
        (2) Customers can edit their collections by clicking on the Edit icon button on the top right of the collection image.






      5. Create NFTs

        Customers can create their own NFTs by clicking on the Create button on the left of their profile icon.





      6. Search

        (1) Customers can see all NFTs minted by collection by going to Search located at the top right
        (2) Here they can search by collection.
        (3) They can then filter by the Blockchain network, Category and Collection

        Customers can sell their minted or owned NFT's (After the admin has verified the NFT in the marketplace) by bidding.


        List the NFT for sale by clicking on the List for Sell button.


        (1) Make sure you are in the "Auction" section.
        (2) Enter the amount you want to sell the NFT for.
        (3) Edit The duration by selecting the start and end dates for the bidding via the calender as shown in the image given below.
        (4) Then press Complete listing button to confirm the listing in the NFT Marketplace.



        (5) Press Confirm button in the Metamask pop-up window to confirm the listing.


      8. List NFT for Sale (Method: Fixed Price Sell)

        Customers can sell their minted or owned NFT's (After the admin has verified the NFT in the marketplace) by a fixed price.


        List the NFT for sale by clicking on the List for Sell button.


        (1) Make sure you are in the "Fixed Sell" section.
        (2) Enter the price you want to sell the NFT for.
        (3) Click on Complete Listing to confirm the listing.


        Press Confirm button in the Metamask pop-up window to confirm the listing.


      9. Customer Logout

        Customers can logout from their profile by clicking on their Profile icon on the top right and then clicking on Log Out button on the left of their profile icon.
